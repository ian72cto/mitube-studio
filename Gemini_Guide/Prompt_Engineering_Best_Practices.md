# 프롬프트 엔지니어링 베스트 프랙티스

성공적인 결과를 얻기 위해 프롬프트에 포함해야 할 6가지 핵심 요소입니다.

## 6가지 핵심 요소
1. **역할(Persona)**: AI에게 특정한 역할(예: 전문 마케터, 소프트웨어 엔지니어)을 부여하세요.
2. **작업(Task)**: 수행해야 할 구체적인 작업을 명시하세요.
3. **맥락(Context)**: 배경 지식이나 상황 설명을 풍부하게 제공하세요.
4. **형식(Format)**: 결과물의 형태(예: 표, 리스트, 이메일 양식)를 지정하세요.
5. **제약 사항(Constraints)**: 제외해야 할 사항이나 지켜야 할 규칙을 정하세요.
6. **예시(Example)**: 원하는 결과와 유사한 예시를 하나 이상 제공하세요.

## 팁
프롬프트가 길어지더라도 상세할수록 AI는 더 풍부하고 정확한 답변을 내놓습니다.
